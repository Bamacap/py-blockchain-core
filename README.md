# py-blockchain-core
A simple, modular blockchain implementation in Python with Proof-of-Work consensus, basic transaction handling, and node communication. Designed for learning and experimentation with blockchain fundamentals.
# New Blockchain

A simple blockchain implementation in Python.

## Features
- Proof-of-Work consensus
- SHA-256 hashing
- Basic transaction handling
- Node communication

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Run: `python main.py`
